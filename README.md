# VLAN-Campus-Network
3 VLAN Campus Network

Project : 3-VLAN Campus Network Implementation


Overview


Designed a VLAN-based campus network to segment HR, Finance, and IT departments using Cisco Packet Tracer.

VLAN Design


VLAN 10 → HR
VLAN 20 → Finance
VLAN 30 → IT

Features

VLAN segmentation
Trunking (802.1Q)
Inter-VLAN routing
Reduced broadcast domain

Tools

Cisco packet tracer

Files

vlan-campus-network.pkt

Outcome

Improved network performance and security.

## Testing
Successfully tested connectivity between VLANs using ping (inter-VLAN routing working).
